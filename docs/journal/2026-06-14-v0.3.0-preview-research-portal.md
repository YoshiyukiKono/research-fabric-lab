# Journal: v0.3.0-preview Research Portal

## 概要

v0.3.0-preview では、Workload Cluster である `agent-lab-02` 上に最小構成の Research Portal を配置する。

この段階では OpenAI API 呼び出しや Experiment Operator 連携は行わない。

目的は、Workload Cluster 上で研究基盤の入口となる Web UI を表示できることを確認することである。

## 構成

```text
agent-lab-02
└─ namespace: research
   └─ research-portal
      ├─ Deployment
      ├─ Service
      ├─ Ingress
      └─ Streamlit App
```

## URL

```text
http://portal.agent.lab.local
```

## 意義

v0.2.0 では Management Cluster 上に MinIO を導入し、Artifact Storage を完成させた。

v0.3.0-preview では、Workload Cluster 側に UI を配置することで、以下の構造が見え始める。

```text
Management Cluster
  Rancher / ArgoCD / MinIO

Workload Cluster
  Research Portal / Runtime Jobs
```
