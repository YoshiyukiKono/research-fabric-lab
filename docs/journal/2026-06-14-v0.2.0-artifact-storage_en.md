# Journal: v0.2.0 Artifact Storage

## Context

After v0.1.0-bootstrap, Research Fabric Lab had:

- Harvester
- Rancher
- ArgoCD
- App of Apps
- Research CRDs
- Runtime image
- Kubernetes Job validation

The next missing layer was artifact storage.

## Decision

Use:

```text
MinIO + PVC + Longhorn
```

as the initial artifact storage layer.

## Why now

Without artifact storage, experiments can run but cannot leave durable research outputs.

Artifact storage establishes the first complete research loop:

```text
Experiment
  -> Runtime Job
  -> Artifact
  -> Storage
```

## Result

v0.2.0 should be considered complete when:

- MinIO is deployed by ArgoCD
- PVC is provisioned by Longhorn
- Ingress is reachable
- `research-artifacts` bucket exists
- Runtime Job uploads `result.json`
