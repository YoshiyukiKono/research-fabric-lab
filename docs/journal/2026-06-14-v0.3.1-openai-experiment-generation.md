
# Journal: v0.3.1 OpenAI Experiment Generation

Date: 2026-06-14

## Summary

Research Fabric Lab において、Research Portal から OpenAI API を利用し、
Research Goal から Experiment Specification を生成することに成功した。

本リリースにより、

Research Idea
→ AI Assistant
→ Experiment Specification

という最初の AI-assisted Research Workflow が実現された。

---

## Background

v0.3.0-preview では、Workload Cluster 上に Streamlit ベースの Research Portal を構築した。

Portal は動作していたが、Experiment 定義は利用者が手動で記述する必要があった。

次の段階として、

Research Goal
↓
OpenAI
↓
Experiment Specification

を生成する機能を追加した。

---

## Architecture

### Management Cluster

rancher-mgmt-02

Responsibilities:

- Rancher
- ArgoCD
- MinIO
- Git Repository Management

### Workload Cluster

agent-lab-02

Responsibilities:

- Research Portal
- Runtime Jobs
- Future Experiment Execution

### OpenAI Integration

Research Portal は Kubernetes Secret として保存された
OpenAI API Key を利用して OpenAI API を呼び出す。

```text
Research Goal
↓
Research Portal
↓
OpenAI API
↓
Experiment JSON
````

---

## Secret Management

API Key は Git Repository に保存しない。

Workload Cluster 上で Secret として管理する。

Example:

```bash
kubectl --context agent-lab \
  -n research \
  create secret generic openai-api-key \
  --from-literal=OPENAI_API_KEY='sk-xxxxxxxx'
```

Portal Deployment では Secret を環境変数として参照する。

## First Generated Experiment

Input:

```text
1次元量子Ising模型の小さなデモ実験を作り、
result.json を生成したい
```

Generated Experiment:

```text
Experiment Name:
ising-1d-demo

Runtime:
ghcr.io/yoshiyukikono/quantum-python-base:latest

Output:
research-artifacts/ising-1d-demo/result.json
```

Generated code:

* 1D Transverse Field Ising Model
* Exact Diagonalization
* Ground State Energy
* Energy Gap
* Magnetization
* ZZ Correlation

The generated experiment was significantly more meaningful than a simple Hello World example.

## Issues Encountered

### OpenAI API Quota

Initial execution failed with:

```text
429 insufficient_quota
```

Reason:

Billing configuration had not yet been enabled.

Resolution:

OpenAI Billing was configured and the request was retried successfully.
